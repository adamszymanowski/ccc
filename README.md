# C++ Crash Course
Code examples from [C++ Crash Course](https://ccc.codes/) book.

I go through them one by one. I already know a bit about C,
but want to learn C++ in an organized manner
and I want to commit to it every day.

## How to compile
Use simple `build.bat` script.

Each chapter has it's own build script for simplicity of use sake.
They should be the same thing, unless I encounter some special case,
as I go through the book.
