# C++ Crash Course
Code examples from [C++ Crash Course](https://ccc.codes/) book.

I go through them one by one. I already know a bit about C,
but want to learn C++ in an organized manner
and I want to commit to it every day.

# Environment
I use Visual Studio 2019.

## How to compile
Use simple `build.bat` script, NOTE: use **x64 Native Tools Command Prompt**, more on that [here](https://github.com/MicrosoftDocs/cpp-docs/blob/master/docs/build/building-on-the-command-line.md).

For example: inside `chapter01` folder run: `build exercise_1_1.cpp`

Each chapter has it's own build script for simplicity of use sake.
They should be the same thing, unless I encounter some special case,
as I go through the book.

## How to run
For example: inside `chapter01` folder run: `builds\exercise_1_1.exe`
